# Running cdktr

Deploy cdktr from single-machine to distributed setups.

## Quick Start

```bash
# Terminal 1: Start principal
cdktr start principal

# Terminal 2: Start agent
cdktr start agent

# Terminal 3: Open TUI
cdktr ui
```
